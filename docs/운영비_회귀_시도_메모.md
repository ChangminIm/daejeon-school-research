# 운영비 격차 회귀 시도 — 별도 메모

> 본 챕터(외부환경분석)는 운영비를 다루지 않음.
> Phase 2 분석 중 시도한 회귀 결과를 별도 메모로만 보관하고,
> 본문/도면에는 노출하지 않음. Phase 3 노선 중복 결과로 효율화 함의만 인용.

작성일: 2026-05-19
관련 산출물: `outputs/tables/운영비격차_회귀결과.csv`

## 1. 시도한 모델

OLS 회귀로 통학차량 운영 12교(흥도초·신탄진용정초 제외)의 **1인당 비용**을 노선·학생 변수로 설명 시도.

- 종속변수: `학생1인당_총비용_천원` (대전_현행통학차량_14개교.csv)
- 독립변수: 노선 총 길이(m), 노선 평균 경사(°), 이용 학생수(명), 차량 대수
- 표본: 운영 12교 중 1인당 비용 데이터 보유 11교 (대전원신흥초 복용분교장 노선 분석 매칭 포함)
- 추정: `statsmodels.OLS` (표준화 X)

## 3. 한계 및 본 챕터 미반영 사유

1. **표본 부족 + 자유도 음수**: 11교에 변수 4개 → Adj R²이 음수로 떨어짐. 회귀 자체가 신뢰 어려움.
2. **outlier 영향 큼**: 구즉초 학생 1명/연 46,600천원, 진잠초 학생 4명/연 11,650천원. 이 두 학교가 학생수 변수의 분모 효과를 왜곡.
3. **혼재 변수**: 차량 대수가 학생수와 강한 상관 — 동일 정보의 부분 대리. 표본 작은 회귀에서 부호 반전을 유발.
4. **운영비는 본 챕터(외부환경분석) 주제가 아님**: 본 챕터는 도시개발·경사 등 외부 환경이 통학환경에 미치는 영향이 주제. 운영비 격차 설명은 별도 챕터(운영현황 분석)의 책무.

## 4. Phase 3 노선 중복 결과로의 인용

운영비 회귀로는 효율화 근거를 정량화하지 못했으나,
**Phase 3 노선 중복 분석**(`outputs/tables/노선중복_쌍별.csv`)에서
공동활용 가능 권역이 도출되면 다음과 같이 본 챕터 결론에 인용 가능:

- "노선 중복률 N% 이상 학교 쌍이 M개 식별 → 공동활용 시 차량 대수 절감 여지"
- 운영비 격차의 직접 회귀가 불가했지만, **공간적 중복 패턴은 운영 효율화의 정책 근거**가 됨

Phase 3 결과는 별도 산출물(`09_노선중복_공동활용권역.png`, `공동활용_후보권역.csv`)로 본 챕터에 노출.
