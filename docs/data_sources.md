## 본 작업에 필요한 외부 데이터 (수집 필요)

### ★★★ 최우선

#### 1. 전국초중등학교위치표준데이터
- 출처: [공공데이터포털](https://www.data.go.kr/data/15021148/standard.do)
- 형식: CSV (위도/경도 컬럼 포함)
- 활용: 학교명으로 송부자료와 join → 정확한 좌표 부여
- 저장 위치: `data/external/전국초중등학교위치표준데이터.csv`

#### 2. 대전 행정구역 경계 SHP
- 출처: SGIS 또는 공공데이터포털 법정동/행정동 경계
- 형식: SHP (시군구 + 행정동 두 레벨)
- 활용: 지도 배경에 경계 표시, 구별 통계 집계
- 저장 위치: `data/external/대전_행정구역.shp` (관련 dbf, shx 등 함께)

#### 3. 재개발/재건축 사업 현황
- 출처:
  - [대전 정비사업관리시스템](https://hreas.daejeon.go.kr)
  - 각 구청 도시정비 부서 (협조 공문 필요할 수 있음)
- 필요 항목: 사업명, 위치 좌표 또는 주소, 세대수, 사업단계, 입주예정일, 시공사
- 저장 위치: `data/external/대전_재개발사업.csv` 또는 .shp

### ★★ 차순위

#### 4. 도시철도 2호선(트램) 자료
- 출처: 대전시 도시철도건설국 (협조 공문)
- 필요 항목:
  - 14개 공구 노선 (선형 SHP)
  - 45개 정거장 위치 (점 SHP)
  - 공구별 공사기간/차로 축소 정보
  - 교통영향평가 자료
- 저장 위치: `data/external/트램_*.shp`

#### 5. 격자 학령인구
- 출처: [통계청 SGIS 통계지리정보](https://sgis.kostat.go.kr)
- 형식: 100m 또는 250m 격자 SHP/CSV
- 활용: 학생 거주지 미제공 보완

#### 6. 시내버스 노선/정류장
- 출처:
  - 대전 BIS (대전교통공사)
  - [국가대중교통DB](https://www.bigdata-transportation.kr)
- 활용: 통학 대체수단 분석

### ★ 추가 가능

#### 7. 도로/보행 네트워크
- 출처: OSM (osmnx 라이브러리로 추출)
- 코드 예시:
  ```python
  import osmnx as ox
  G = ox.graph_from_place("Daejeon, South Korea", network_type="walk")
  ```

#### 8. TAAS 교통사고 자료
- 출처: 교통사고분석시스템 TAAS
- 활용: 어린이보호구역 사고다발지 분석

#### 9. 학교 통학구역 폴리곤
- 출처: 대전시교육청 행정과 또는 동부/서부교육지원청
- 있으면 매우 유용 (거주지 데이터 미제공의 큰 보완)

#### 10. 장래 학령인구 추계
- 출처: 통계청 KOSIS, 대전세종연구원

## 데이터 라이선스/이용 주의
- 공공데이터포털 데이터: 출처 표시 후 자유 이용 (대부분 OGL)
- SGIS 통계지리정보: 비상업적 이용 가능
- 정부/지자체 협조자료: 연구목적 한정, 외부 배포 금지일 수 있음
- 개인정보 포함 자료(거주지 등)는 절대 외부 배포 금지

## 데이터 추가 후 작업
1. `data/external/`에 파일 저장
2. `data/external/README.md`에 출처/수집일/버전 메모
3. 필요시 `src/`의 관련 모듈 업데이트
4. `python run.py`로 파이프라인 재실행
