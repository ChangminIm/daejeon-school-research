## 2. Phase B-1 — 신규 아파트 경사도 가설 검정

### 검정 결과
- **A vs C**: Mann-Whitney U = 1900, **p = 0.1996** (정규성 미충족, 비모수)

### 해석
- A·B·C 그룹 모두 대전 평균(13.7°)의 1/3 수준 — 시가지·학교·재개발 모두 평지에 위치

- 가설 기각: 대전에서 "신규 아파트 = 가파름"은 성립하지 않음

## 4. Phase B-2: 4변수 로지스틱 회귀 결과

### v1 (14교 일괄 기준) — 초기 분석, 참고용
종속변수가 14교(운영 12+예정 1+명목 1) 일괄이어서 흥도초·신탄진용정초의 노이즈가 섞임.
- 표본: 243교 (y=1: 14교, y=0: 229교)
- β_경사도 = +0.57, p = 0.080 (유의 미달)
- AIC 75.73, Pseudo R² 0.416

### v2 (운영 12교 기준) — 정정 후 본 결과
실제 운영 학교만 y=1로 분리하여 재실행.

### 모델 설계
- 종속변수: 현행 운영 12교 여부 (1/0). 흥도초(예정)·신탄진용정초(명목)는 표본 제외
- 독립변수: 학생수, 도심거리(km), 학교밀도(반경 2km), 경사도(slope_300m_mean)
- 표본: 241교 (y=1: 13건 [기성초·길헌분교 분리 포함], y=0: 228교)
- 추정: statsmodels Logit (MLE), 변수는 모두 표준화

### v2 모델 적합도

| 지표 | 값 |
|---|---:|
| AIC | **61.10** |
| BIC | 78.53 |
| Pseudo R² | **0.495** |
| Log-Likelihood | −25.55 |
| 수렴 | True |

v1 → v2 비교: AIC 75.73 → 61.10 (Δ −14.6, 큰 폭 개선), Pseudo R² 0.416 → 0.495.
운영 12교 분리만으로 모델 적합도가 큰 폭 향상.

### v2 4변수 계수표 (표준화)

| 변수 | β | SE | z | p-value | 95% CI |
|---|---:|---:|---:|---:|---|
| (절편) | −5.444 | 0.987 | −5.52 | <0.001 | [−7.38, −3.51] |
| 학생수 | −0.038 | 0.484 | −0.08 | 0.938 | [−0.99, +0.91] |
| 도심거리 | **+1.104** | 0.501 | 2.20 | **0.028 ★** | [+0.12, +2.09] |
| 학교밀도 | −1.509 | 0.860 | −1.76 | 0.079 · | [−3.19, +0.18] |
| **경사도** | **+0.777** | 0.370 | 2.10 | **0.035 ★** | [+0.05, +1.50] |

- **경사도 p = 0.035로 통상 유의수준(0.05) 충족** (v1 p=0.080 → v2 0.035)
- 도심거리도 유의 (p=0.028, β +1.10)
- 학교밀도는 한계 유의 (p=0.079, β −1.51)
- 학생수는 무의미 (p=0.938)

### 매칭률 (sklearn LogisticRegression)
- 확률 ≥ 0.5 기준 운영 12교 예측: **6/13 (46.2%)**
- 확률 상위 12교 ∩ 실제 운영 12교: **8/12 (66.7%)**

### 미반영 결정 근거 (v2 정정)

v2 회귀에서 경사도가 통계적으로 유의해졌으나(β=+0.777, p=0.035), 다음 사유로 점수 산식 미반영 결정은 유지:

1. ~~통계적 유의성 부족~~ → **v2에서 충족됨**(p=0.035). v1의 비유의성은 흥도초·신탄진용정초 노이즈가 원인이었음을 확인.
2. **정책 시뮬레이션 불일치 (가장 중요)**: 경사도 반영 시 정책 대상인 외곽·신규 아파트권 학교가 상위 30위에서 빠지고, 본 용역에서 지원 의향이 없는 도심·중학교가 새로 진입함 (시뮬레이션 6교 중 5교가 중학교)
3. **모델 단순성 유지**: 추가 지표는 외부 보고서·발표 시 설명 부담 증가
4. **본 정책 방향과의 정합성**: 운영 12교 패턴의 경사 신호는 외곽도·학교밀도와 정보가 중첩(VIF<5이지만 부호 상호의존) — 별도 가중치의 정책적 부가가치 제한적
5. **운영 분류 도입으로 변수 추가 대신 표기 개선**: 흥도초(예정)·신탄진용정초(명목) 별도 표기로 14교 일괄 표시의 데이터 노이즈를 해소

### 14교 사후 검증 지표로의 활용

경사도는 점수 산식에 미반영하되, 본 분석의 **사후 검증 자료**로 활용:

- 14교 평균 경사 8.05° vs 나머지 학교 4.36° (차이 +3.69°, Mann-Whitney p<0.0001)
- 본 분석의 점수가 식별한 "외곽 산악 학교" 결과의 데이터적 정합성 확인
- 학교 팝업의 보조 정보로 표시 (대전 평균·14교 평균과 비교)

## 5. Phase C — 점수 반영 결정

### 분기 기준 (사전 정의)
- |β| ≥ 0.3 → 점수 반영 + 가중치 재산정 (Phase D 실시)
- |β| < 0.3 → 시각화/팝업만 (Phase D 미실시)

### 결정: **점수 미반영**

v1(14교 일괄) β = +0.57, p = 0.080 → v2(운영 12교 기준) β = +0.777, p = 0.035로 임계값 모두 충족.
그럼에도 다음 사유를 종합하여 **점수 산식 미반영**으로 최종 결정 (상세는 §4 "미반영 결정 근거" 참조):

1. **정책 시뮬레이션 불일치 (가장 중요)**: 반영 시 정책 대상 외곽·신규 아파트권 6교가 상위 30위에서 이탈, 도심·중학교 5교 진입
2. 모델 단순성 유지 (4지표 → 3지표 + 보조 정보 노출)
3. 운영 분류 도입(운영 12 / 예정 1 / 명목 1)으로 14교 일괄 표시의 노이즈를 표기 차원에서 해소 — 변수 추가 대안

즉 본 결정의 핵심 근거는 "정책 시뮬레이션 불일치"이며, 통계적 유의성·다중공선성은 v2 결과 모두 임계 통과.

### Phase D — 적격성 점수 재산출
**미실시**. 가중치는 다음 그대로 유지:
- 동급학교밀도 0.50
- 도심거리 0.30
- 학교규모 0.20

## 7. 보고서 본문 초안

> 통학지원 적격성 산식은 동급학교밀도, 도심거리, 학교규모 세 지표를 가중합하여 산출되며, 경사도는 직접 변수로 포함되지 않는다. 그러나 본 연구는 산식의 외부 검증을 위해 학교 부지 반경 300m의 평균 경사도를 5m 해상도 지형 데이터로 별도 추출하여 비교 분석을 수행하였다.
>
> 분석 결과 현행 통학버스 운영 14교의 평균 경사도는 8.05°로, 비운영 학교 평균 4.36°보다 약 3.69° 가팔랐으며, 이 차이는 Mann-Whitney U 검정에서 매우 강한 유의성을 보였다(p < 0.001). 이는 기존 산식이 "동급학교밀도"와 "도심거리" 지표를 통해 외곽 산지 학교의 통학 부담을 간접적으로 포착하고 있음을 시사한다.
>
> 한편, 일반적으로 제기되는 "신규 대규모 주거단지는 가파른 부지에 들어선다"는 가정은 본 분석에서 기각되었다. 재개발 추진 중인 공사중·관리처분 단계 단지(n=19)의 평균 경사도는 3.43°로, 학교 평균 4.58°와 통계적으로 유의한 차이를 보이지 않았다(p = 0.20). 대전의 재개발 부지는 대체로 기존 시가지 평지에 분포하므로, 경사도가 신규 통학 수요를 발생시키는 별도 요인으로 작용하지는 않는 것으로 판단된다.
>
> 종합하여 본 연구는 경사도를 산식의 추가 지표로 반영하지 않되, 산식의 사후 검증 자료 및 학교별 보조 정보로 활용한다.
