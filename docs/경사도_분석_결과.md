## 2. Phase B-1 — 신규 아파트 경사도 가설 검정

### 가설
"재개발 등 신규 대규모 주거단지는 (학교가 위치한 평지보다) 경사가 급한 곳에 들어서므로, 통학환경에 추가 부담이 된다"

### 그룹 정의
| 그룹 | 정의 | n | 측정 컬럼 |
|---|---|---|---|
| A 신규 아파트 | 재개발 중 (공사중 + 관리처분) | 19 | slope_500m_mean |
| B 진행 전체 | 1·2·3·4·5단계 (완료/입안/미정 제외) | 73 | slope_500m_mean |
| C 학교 245교 | 대전 초·중학교 전수 | 243 | slope_300m_mean |
| D 대전 영역 | 5m 래스터 random sample | 1,000 | 픽셀값 |

### 기술 통계 (단위 °)
| 그룹 | mean | median | std |
|---|---|---|---|
| A 신규 아파트 | **3.43** | 2.95 | 1.88 |
| B 진행 전체 | 3.92 | 3.33 | 2.59 |
| C 학교 245교 | **4.58** | 3.79 | 3.12 |
| D 대전 영역 | 13.67 | 11.24 | 11.91 |

### 검정 결과
- **A vs C**: Mann-Whitney U = 1900, **p = 0.1996** (정규성 미충족, 비모수)
  - Welch t-test(참고용): t = -2.43, p = 0.022
- **A vs B**: U = 668.5, **p = 0.8132**

### 해석
- A·B·C 그룹 모두 대전 평균(13.7°)의 1/3 수준 — 시가지·학교·재개발 모두 평지에 위치
- 신규 아파트가 학교보다 -1.16° **더 평탄** (방향은 가설 반대)
- 가설 기각: 대전에서 "신규 아파트 = 가파름"은 성립하지 않음

## 4. Phase B-2: 4변수 로지스틱 회귀 결과

### 모델 설계
- 종속변수: 현행 통학차량 운영 14교 여부 (1/0)
- 독립변수: 학생수, 도심거리(시청 기준 km), 동급학교밀도(반경 2km), 경사도(slope_300m_mean)
- 표본: 학교 243교 (초 154 + 중 89)
- 추정: statsmodels Logit (MLE), 변수는 모두 표준화

### 다중공선성 진단 (VIF)

| 변수 | VIF |
|---|---:|
| 학생수 | 2.58 |
| 도심거리 | 3.92 |
| 동급학교밀도 | 2.26 |
| 경사도 | 3.20 |

→ 모든 변수 VIF < 5. 일반 기준상 다중공선성 "심각" 단계는 아님. 다만 변수 간 상관관계가 존재하여 4변수 회귀에서 학생수 계수가 3변수 모델 대비 부호 반전(−0.138 → +0.103, p=0.80) — 정보 중복 신호.

### 모델 비교

| 모델 | AIC | BIC | Pseudo R² |
|---|---:|---:|---:|
| 3변수 (경사 제외) | 76.78 | 90.76 | 0.389 |
| 4변수 (경사 포함) | 75.73 | 93.19 | 0.416 |

- ΔAIC = −1.06 (통상 ΔAIC ≥ 2가 모델 개선 의미 있음)
- BIC는 오히려 증가 (+2.43) — 모델 복잡성 패널티 고려 시 3변수가 더 좋음
- 우도비 검정: χ²=3.06, p=0.080 → 경사도 추가가 통계적으로 유의하지 않음

### 4변수 모델 계수표 (표준화)

| 변수 | β | SE | z | p-value | 95% CI |
|---|---:|---:|---:|---:|---|
| (절편) | −4.59 | 0.73 | −6.28 | <0.001 | [−6.02, −3.15] |
| 학생수 | +0.10 | 0.41 | 0.25 | 0.801 | [−0.70, +0.90] |
| 도심거리 | +1.17 | 0.44 | 2.68 | 0.007 | [+0.31, +2.02] |
| 동급학교밀도 | −1.08 | 0.70 | −1.55 | 0.120 | [−2.45, +0.28] |
| **경사도** | **+0.57** | **0.33** | **1.75** | **0.080** | **[−0.07, +1.22]** |

- |β_경사도| = 0.57 > 0.3 임계값 **충족**
- 단, p = 0.080으로 통상 유의수준(0.05) 미달
- 95% 신뢰구간이 0을 포함 → 효과 방향 확정 불가

### 정책 시뮬레이션: 가중치별 상위 30위 변화

경사도를 점수에 반영했을 때(가중치 0.15) 미운영 학교 상위 30위 구성 변화:

| 학하초 | 유성구 | 156 | 6.7 km | 1.8° |
| 신탄진초 | 대덕구 | 263 | 11.5 km | 2.2° |

### 미반영 결정 근거

본 분석은 다음 4가지 사유로 경사도를 점수 산식에 반영하지 않음:

1. **통계적 유의성 부족**: 경사도 단독 추가의 통계적 의미가 약함 (LR 검정 p=0.080)
2. **모델 개선 효과 미미**: ΔAIC = −1.06로 통상 임계값(−2) 미달, BIC는 오히려 악화
3. **정책 시뮬레이션 불일치**: 경사도 반영 시 정책 대상인 외곽·신규 아파트권 학교가 상위 30위에서 빠지고, 본 용역에서 지원 의향이 없는 도심·중학교가 새로 진입함 (6/6교 중 5교 중학교)
4. **본 정책 방향과의 정합성**: 14교 패턴의 경사 신호는 외곽도·학교밀도와 정보가 중첩되어 별도 가중치 부여의 정책적 부가가치가 부재함

### 14교 사후 검증 지표로의 활용

경사도는 점수 산식에 미반영하되, 본 분석의 **사후 검증 자료**로 활용:

- 14교 평균 경사 8.05° vs 나머지 학교 4.36° (차이 +3.69°, Mann-Whitney p<0.0001)
- 본 분석의 점수가 식별한 "외곽 산악 학교" 결과의 데이터적 정합성 확인
- 학교 팝업의 보조 정보로 표시 (대전 평균·14교 평균과 비교)

## 5. Phase C — 점수 반영 결정

### 분기 기준 (사전 정의)
- |β| ≥ 0.3 → 점수 반영 + 가중치 재산정 (Phase D 실시)
- |β| < 0.3 → 시각화/팝업만 (Phase D 미실시)

### 결정: **점수 미반영**

실제 회귀 결과 |β_경사도| = 0.57로 사전 임계값은 충족했으나, 다음 4가지 사유를 종합하여 **점수 산식 미반영**으로 최종 결정 (상세는 §4 "미반영 결정 근거" 참조):

1. 통계적 유의성 부족 (LR 검정 p = 0.080)
2. 모델 개선 효과 미미 (ΔAIC = −1.06, BIC 악화)
3. 정책 시뮬레이션 불일치 (반영 시 정책 대상 외곽·신규 아파트권 6교가 상위 30위에서 이탈, 도심·중학교 5교 진입)
4. 본 용역의 정책 방향과 정합성 부족

즉 본 결정의 핵심 근거는 "통계적 유의성 부족"과 "정책 시뮬레이션 결과 불일치"이며, 다중공선성 자체는 부차적 신호 (모든 변수 VIF<5).

### Phase D — 적격성 점수 재산출
**미실시**. 가중치는 다음 그대로 유지:
- 동급학교밀도 0.50
- 도심거리 0.30
- 학교규모 0.20

## 7. 보고서 본문 초안

> 통학지원 적격성 산식은 동급학교밀도, 도심거리, 학교규모 세 지표를 가중합하여 산출되며, 경사도는 직접 변수로 포함되지 않는다. 그러나 본 연구는 산식의 외부 검증을 위해 학교 부지 반경 300m의 평균 경사도를 5m 해상도 지형 데이터로 별도 추출하여 비교 분석을 수행하였다.
>
> 분석 결과 현행 통학버스 운영 14교의 평균 경사도는 8.05°로, 비운영 학교 평균 4.36°보다 약 3.69° 가팔랐으며, 이 차이는 Mann-Whitney U 검정에서 매우 강한 유의성을 보였다(p < 0.001). 이는 기존 산식이 "동급학교밀도"와 "도심거리" 지표를 통해 외곽 산지 학교의 통학 부담을 간접적으로 포착하고 있음을 시사한다.
>
> 한편, 일반적으로 제기되는 "신규 대규모 주거단지는 가파른 부지에 들어선다"는 가정은 본 분석에서 기각되었다. 재개발 추진 중인 공사중·관리처분 단계 단지(n=19)의 평균 경사도는 3.43°로, 학교 평균 4.58°와 통계적으로 유의한 차이를 보이지 않았다(p = 0.20). 대전의 재개발 부지는 대체로 기존 시가지 평지에 분포하므로, 경사도가 신규 통학 수요를 발생시키는 별도 요인으로 작용하지는 않는 것으로 판단된다.
>
> 종합하여 본 연구는 경사도를 산식의 추가 지표로 반영하지 않되, 산식의 사후 검증 자료 및 학교별 보조 정보로 활용한다.
